# 💰 Loan Approval Prediction System

A Machine Learning web application that predicts whether a loan application will be approved or rejected based on applicant details and financial information.

## 🚀 Project Overview

This project uses a CatBoost Classifier to predict loan approval status. The model was trained on a loan approval dataset and achieved approximately **99.3% accuracy**.

The application is built using **Python**, **Machine Learning**, and **Streamlit**.

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/kumarhimanshu97834-art/Loan-Approval-Prediction.git
```

Move into the project directory:

```bash
cd Loan-Approval-Prediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app.py
```
